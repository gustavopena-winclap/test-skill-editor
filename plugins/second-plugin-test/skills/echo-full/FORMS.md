# Echo Full — Input Form

Collect these before running the script.

| field  | type   | required | default | notes                        |
|--------|--------|----------|---------|------------------------------|
| text   | string | yes      | —       | the payload to echo          |
| mode   | string | no       | `plain` | see REFERENCE.md             |
| repeat | int    | no       | `1`     | how many times to print it   |

Map to flags: `--text`, `--mode`, `--repeat`.
