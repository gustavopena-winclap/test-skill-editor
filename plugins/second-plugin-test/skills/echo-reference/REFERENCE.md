# Echo Transform Reference

Modes accepted for `mode`:

| mode    | effect                          | example (`hi`) |
|---------|---------------------------------|----------------|
| plain   | no change                       | `hi`           |
| upper   | uppercase                       | `HI`           |
| reverse | reverse the characters          | `ih`           |
| shout   | uppercase + `!!!` suffix        | `HI!!!`        |

Default mode: `plain`.
