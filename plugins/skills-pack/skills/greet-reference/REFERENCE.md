# Greeting Style Reference

Lookup table of greeting templates. `<name>` is replaced with the input name.

| style    | template                                  |
|----------|-------------------------------------------|
| casual   | `Hey <name>! 👋`                          |
| formal   | `Good day, <name>. A pleasure to meet you.` |
| pirate   | `Ahoy, <name>! Welcome aboard, matey. 🏴‍☠️` |
| robot    | `GREETINGS <name>. BEEP BOOP.`            |

Default style: `casual`.
